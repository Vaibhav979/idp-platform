# 🔌 API Documentation (Day 1)

## 📦 Create Project

POST /projects

### Request Body

{
"name": "my-app",
"repoUrl": "https://github.com/user/repo"
}

### Response

{
"id": 1,
"name": "my-app",
"repoUrl": "...",
...
}

---

## 🚀 Trigger Deployment

POST /deployments/{projectId}

### Response

{
"id": 1,
"status": "PENDING"
}

---

## 📊 Get Deployment

GET /deployments/{id}

### Response

{
"id": 1,
"status": "RUNNING"
}
