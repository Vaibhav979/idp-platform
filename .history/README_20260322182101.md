# 🚀 Internal Developer Platform (IDP)

A backend system that allows users to deploy applications from Git repositories, similar to platforms like Heroku or Render.

## ✨ Features (Day 1)

* Create Projects
* Trigger Deployments
* Track Deployment Status (simulated pipeline)

## 🏗️ Architecture (Day 1)

* Spring Boot backend
* PostgreSQL (or H2)
* REST APIs
* Async deployment simulation

## 📦 Core Entities

* User
* Project
* Deployment

## Deployment Lifecycle

PENDING → BUILDING → DEPLOYING → RUNNING / FAILED

## 🚀 APIs

### Create Project

POST /projects

### Trigger Deployment

POST /deployments/{projectId}

### Get Deployment

GET /deployments/{id}

## How to Run

```bash
./mvnw spring-boot:run
```

## Vision

This project aims to evolve into a full-fledged Internal Developer Platform (IDP) with CI/CD, GitOps, Kubernetes orchestration, and observability.
